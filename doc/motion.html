Motion

Selectable positions

Neomacs try to keep the focus at selectable positions after every command invocation.

fundoc'selectable-p
fundoc'selectable-p-aux
fundoc'adjust-marker-direction
fundoc'ensure-selectable

Motion by nodes and elements

fundoc'forward-node
fundoc'backward-node
fundoc'forward-node-cycle
fundoc'backward-node-cycle
fundoc'forward-element
fundoc'backward-element
fundoc'backward-up-node

Top-level motion

fundoc'beginning-of-defun
fundoc'end-of-defun
fundoc'beginning-of-buffer
fundoc'end-of-buffer

Word motion

fundoc'word-boundary-list
fundoc'word-character-p
fundoc'word-start-p
fundoc'word-end-p

Word motion commands:

fundoc'forward-word
fundoc'backward-word

Line motion

Test if position is at line boundary:

fundoc'block-element-p
fundoc'block-element-p-aux
fundoc'line-start-p
fundoc'line-end-p

Move to line boundary:

fundoc'beginning-of-line
fundoc'end-of-line

The following commands try to keep horizontal location approximately the same. Currently, we do this by counting the number of selectable positions between current focus position and beginning-of-line position, and try to keep that number the same.

fundoc'previous-line
fundoc'next-line
fundoc'scroll-lines
fundoc'scroll-up-command
fundoc'scroll-down-command
