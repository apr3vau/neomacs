Buffer focus

Neomacs keep track of the current position being edited using a focus marker.

fundoc'focus

Run some action when buffer focus is moved:

fundoc'on-focus-move

Customize how buffer focus is rendered:

fundoc'render-focus-aux

Buffer selection

A range of Neomacs buffer content can be selected. Neomacs selection is tracked with a selection marker. The range between the selection and focused marker is considered selected when selection-active is t. Certain commands (for example, some clipboard commands) that normally act on focused element will act on the selected range instead if selection is active.

fundoc'selection-marker
fundoc'selection-active
